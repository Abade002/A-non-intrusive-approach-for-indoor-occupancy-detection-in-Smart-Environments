data0_temp.mat: Temperature Data Analysis
-tempwi: Temperature with occupants, 1st column Indoor Temperature, 2st column Outdoor Temperature
-tempwo: Temperature without occupants, 1st column Indoor Temperature, 2st column Outdoor Temperature

data0_co2.mat: Carbon Dioxide Data Analysis
-co_0: Carbon Dioxide without occupants
-co_1: Carbon Dioxide with one occupant
-co_2: Carbon Dioxide with more than one occupant
-nroom_1: Number of occupants with one occupant data

data0_noise.mat:
-noise_wi: Noise with occupants
-nroom_wi: Noise without occupants
-nroom_wi: Number of occupants with occupants data

data0_lux.mat: Lux Data Analysis
-lux_wi: Lux with occupants
-lux_wo: Lux without occupants
-nroom_wi: Number of occupants with occupants data
